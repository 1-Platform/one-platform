## Install

```sh
npm install
```

## Usage

### Development server

- Run development server

```sh
ng [serve](https://angular.io/cli/serve)
```

### Build

```sh
ng [build](https://angular.io/cli/build)
```

## Run tests and lint

```sh
ng [lint](https://angular.io/cli/lint)
ng [test](https://angular.io/cli/test)
```
